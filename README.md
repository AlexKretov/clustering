## ⚙️ Используемые методы

### Предобработка
- `Label Encoding` категориальных признаков
- `StandardScaler` для нормализации

### 🤖 Алгоритмы кластеризации

| Алгоритм | Метод подбора параметров |
|----------|--------------------------|
| **K-Means** | Elbow Method + Silhouette Analysis |
| **Agglomerative Hierarchical Clustering** | Ward linkage |
| **DBSCAN** | Метод k-расстояний для подбора `eps` |
| **Gaussian Mixture Models (GMM)** | BIC/AIC + компромисс с силуэтом |

### 📉 Снижение размерности и визуализация
`PCA` · `t-SNE` · `UMAP`

### 📦 Библиотеки
```
pandas · numpy · scikit-learn · matplotlib · seaborn · umap-learn · tqdm
```

## 📈 Основные результаты

> **Лучший алгоритм по интерпретируемости — GMM с 4 компонентами**

| Алгоритм | Результат |
|----------|-----------|
| 🟢 **K-Means** | Наилучший силуэт при **k = 6** |
| 🟡 **Hierarchical Clustering** | Выделил **2 кластера** по силуэту |
| 🔴 **DBSCAN** | Не смог выделить более **1 кластера** |
| 🟣 **GMM** | **4 компоненты** — хорошая интерпретируемость, эллипсоидальные кластеры с оценкой вероятностей принадлежности |

> 💡 Все модели сохраняют метки в финальном датафрейме для дальнейшего анализа профилей клиентов

## 🔍 Выводы

GMM выделил **4 статистически значимых сегмента**, различающихся по социально-демографическим и кредитным характеристикам.

**Ключевые разделители:** статус иностранного работника · наличие поручителей · количество иждивенцев · размер кредитной сделки

### 👥 Профили сегментов

<details>
<summary><b>🟥 Кластер 0 — «Трудовые мигранты-арендаторы»</b> (высокорисковый массовый сегмент)</summary>

<br>

**Маркер:** `foreign_worker = 0.97` (против 0.04 в среднем по портфелю)

**Кредитный профиль:**
- Минимальные суммы: **2 690 у.е.** (−0.21σ)
- Короткие сроки: медиана ≈ **12 мес.**
- Отсутствие собственности, преимущественно арендное жильё

**⚠️ Риски:** высокая мобильность, отсутствие залогового обеспечения, нестабильность трудового статуса

**💼 Бизнес-ценность:** низкий чек, высокая частота обращений, низкая кросс-продажная ёмкость

</details>

<details>
<summary><b>🟨 Кластер 1 — «Социально обеспеченные заёмщики»</b> (средний риск)</summary>

<br>

**Маркер:** `Other_debtors_guarantors = 1.55` (z=+2.94) — наличие поручителя/созаёмщика

**Кредитный профиль:**
- Средние суммы: **≈ 3 000 у.е.**
- Стандартные сроки, нейтральная кредитная история

**Особенность:** сделка поддержана третьими лицами (родители, супруги)

**⚠️ Риски:** зависимость от платёжеспособности поручителя

**✅ Возможности:** низкий удельный вес дефолта благодаря социальному контролю

</details>

<details>
<summary><b>🟩 Кластер 2 — «Семьи с иждивенцами»</b> (прибыльный сегмент роста)</summary>

<br>

**Маркер:** `Number_of_people_liable = 2.0` (z=+2.33) — два иждивенца (модель «ребёнок + супруг»)

**Кредитный профиль:**
- Максимальные суммы: **3 467 у.е.** (+0.07σ)
- Стабильный трудовой стаж, целевые кредиты (техника, образование, ремонт)

**⚠️ Риски:** высокая долговая нагрузка, чувствительность к кризисам

**✅ Возможности:** высокий потенциал кросс-продаж (ипотека, страхование, накопительные продукты)

</details>

<details>
<summary><b>🟦 Кластер 3 — «Стабильные собственники»</b> (низкий риск, премиум)</summary>

<br>

**Маркер:** возраст 40+ лет, наличие недвижимости, длительные сроки кредитования

**Кредитный профиль:**
- Широкий диапазон сумм (до **18 000 у.е.**)
- Сроки **> 36 мес.**, низкая доля иждивенцев

**✅ Риски:** минимальны; высокая лояльность клиента

**⚠️ Особенность:** «красные точки» (низкая уверенность GMM) — граничные случаи с Кластером 2, требующие ручной верификации

</details>

### 🎯 Стратегические рекомендации

| Сегмент | 🛡️ Управление рисками | 📣 Маркетинг & Retention |
|---------|----------------------|--------------------------|
| **Кластер 0** 🟥 | Требования к трудовому договору (срок > срока кредита); микро-залоги; мониторинг миграционного статуса | Массовые предложения с быстрым одобрением; SMS-сервис; продукты для переводов за рубеж |
| **Кластер 1** 🟨 | Оценка платёжеспособности поручителя; юридическая чистота договоров | Рефинансирование по сниженной ставке при наличии поручителя; семейные пакеты |
| **Кластер 2** 🟩 | Жёсткий контроль PTI; обязательное страхование; лимитирование суммы по иждивенцам | Кросс-селлинг ипотеки и накопительных счетов; сезонные предложения |
| **Кластер 3** 🟦 | Упрощённая схема одобрения (express-скоринг); преференциальные ставки за лояльность | VIP-обслуживание; премиальные карты; консьерж-сервис; инвестиционные продукты |
